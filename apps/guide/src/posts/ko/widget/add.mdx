# iPhone 위젯 추가 방법

## iPhone 위젯 추가 방법

### iPhone 위젯 추가 방법

1. 아이폰 홈 화면의 빈 영역 혹은 앱 사이 공간을 앱들이 '흔들릴 때'까지 길게 누릅니다.

2. 왼쪽 상단에 나타난 + 버튼을 누릅니다.

3. 위젯을 추가하려는 앱을 검색합니다. 예시 이미지에서는 <NewTab href="https://apps.apple.com/kr/app/id6467635137">스케줄러</NewTab> 앱을 검색하고 있습니다.

4. 원하는 위젯을 선택한 뒤 '위젯 추가' 버튼을 누릅니다.

5. 이어서 다른 위젯을 추가하려면 다시 + 버튼을 눌러 2~4를 반복합니다.

6. 위젯 추가를 완료했다면 완료 버튼을 누르거나, 빈 공간을 눌러 흔들리는 앱들을 원래대로 돌려놓을 수 있습니다.

```swift
func greet() {
print("Hello, world!")
}
```
